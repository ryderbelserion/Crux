## Contributing
Contributing to Crux is the easiest thing in the world!
<Steps>
1. git clone https://github.com/ryderbelserion/Crux.git
2. pnpm install
3. pnpm dev
4. Edit files in `docs`, It will refresh immediately with new changes
</Steps>
