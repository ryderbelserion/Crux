## Commands
This page lists all the permissions available for ChatterBox

| Permission                | Command   | Description                |
|---------------------------|-----------|----------------------------|
| chatterbox.command.reload | cb reload | Gives access to /cb reload |
| chatterbox.command.motd   | cb motd   | Gives access to /cb motd   |
| chatterbox.command.use    | cb        | Gives access to /cb        |

### Color Formatting
| Permission            | Description                                                             |
|-----------------------|-------------------------------------------------------------------------|
| chatterbox.color      | Allows use of standard colors, such as <red> or <yellow> in chat.       |
| chatterbox.gradient   | Allows use of gradient colors, such as `<gradient:red:yellow>` in chat. |
| chatterbox.rainbow    | Allows use of rainbow colors, like <rainbow>                            |
| chatterbox.font       | Allows use of rainbow colors, like <rainbow>                            |

### Decoration Formatting
| Permission                          | Description                            |
|-------------------------------------|----------------------------------------|
| chatterbox.decoration               | Allows use of all decoration tags.     |
| chatterbox.decoration.bold          | Allows use of <bold> in chat.          |
| chatterbox.decoration.italic        | Allows use of <italic> in chat.        |
| chatterbox.decoration.underlined    | Allows use of <underlined> in chat.    |
| chatterbox.decoration.strikethrough | Allows use of <strikethrough> in chat. |
| chatterbox.decoration.obfuscated    | Allows use of <obfuscated> in chat.    |
