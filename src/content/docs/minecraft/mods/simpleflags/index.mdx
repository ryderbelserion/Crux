# SimpleFlags

### Installation
#### Requirements
:::tip
SimpleFlags requires you to use at least **Java 25**, and Paper **26.1.X**
:::

<Steps>
1. Grab the newest version of SimpleFlags
2. Add it to the `plugins` folder
3. Restart, or start your Paper Server!
</Steps>
