## Commands
This page lists all the permissions available for SimpleFlags

| Permission         | Command            | Description                         | Default |
|--------------------|--------------------|-------------------------------------|---------|
| simpleflags.reload | simpleflags reload | Gives access to /simpleflags reload | OP      |
| simpleflags.help   | simpleflags help   | Gives access to /simpleflags motd   | OP      |
| simpleflags.access | simpleflags        | Gives access to /simpleflags        | OP      |
