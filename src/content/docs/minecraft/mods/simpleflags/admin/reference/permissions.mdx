**This page lists all permissions included in SimpleFlags.**

<Aside type="tip">
    The base command `/simpleflags` requires `simpleflags.access`
</Aside>

### Command Permissions
| Command               | Permission Node    | Description                         | Player has by default? |
|-----------------------|--------------------|-------------------------------------|------------------------|
| `/simpleflags reload` | simpleflags.help   | Gives access to /simpleflags reload | op                     |
| `/simpleflags help`   | simpleflags.reload | Gives access to /simpleflags help   | op                     |
